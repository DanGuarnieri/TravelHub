# TravelHub

## Visão Geral

TravelHub é uma plataforma web desenvolvida para centralizar todo o planejamento, execução e registro de viagens.

O objetivo é oferecer uma experiência semelhante a um assistente pessoal de viagens, permitindo ao usuário:

- Planejar viagens
- Controlar orçamento
- Organizar documentos
- Criar roteiros
- Gerenciar checklists
- Armazenar fotos
- Utilizar recursos de Inteligência Artificial
- Gerar resumos automáticos das viagens

---

# Objetivos do Projeto

Centralizar em um único sistema:

- Planejamento
- Controle financeiro
- Organização documental
- Diário de viagem
- Memórias fotográficas
- Apoio por IA

O projeto está sendo desenvolvido como estudo prático de:

- Django
- Arquitetura MVC/MVT
- Banco de Dados Relacional
- Integração com IA
- Desenvolvimento Full Stack
